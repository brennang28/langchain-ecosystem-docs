Deep Dives

Under the Hood

Expand each section to explore the core concepts of every product in plain English.

Models

"The brain — different AI models you can plug in (GPT, Claude, Gemini). Pick the right brain for the job."

Messages

"The conversation history — what you said, what the AI said. Context is everything."

Tools

"Extensions — like giving the AI a calculator, web browser, or database access."

Agents

"A smart assistant that decides which tools to use based on your question."

Streaming

"Getting the answer word-by-word in real time instead of waiting for the whole thing."

Structured Output

"Getting answers in a specific format (JSON, tables) instead of free text."

Graph

"A flowchart that your AI follows — defining the overall process from start to finish."

Nodes

"Each box in the flowchart — a step or decision point where something happens."

Edges

"The arrows connecting steps — the flow of logic that determines what happens next."

State

"A shared notepad passed between steps so they can share information."

Persistence

"Saving the notepad to disk so you can pause and resume your workflow later."

Interrupts

"Pause buttons — the AI stops and asks a human before continuing."

Subagents

"Specialists the main agent hires for specific tasks — one for research, one for code, one for writing."

Skills

"Pre-packaged expertise — like hiring someone who already knows accounting or data analysis."

Sandboxes

"Secure playgrounds where agents can run code safely without breaking anything."

Memory

"The agent's notebook — remembering past conversations, facts, and preferences."

Streaming

"Watching the agent work in real time — seeing its progress, thoughts, and outputs as they happen."

Human-in-the-Loop

"A checkpoint where the agent asks you 'Is this right?' before proceeding with critical actions."
